EIZEN_EXERCISES

This repository features my completed front-end projects for the Software Engineering assessment.


Exercise Folders

- 01-sementic-markup - A recipe page built with strict HTML structural tags.

- 02-css-styling - A single-page article designed with a separate stylesheet and CSS variables.

- 03-page-layout - A responsive image and video gallery.

- 04-web-accessibility - A fully accessible blog layout featuring keyboard navigations.


Notes

- Every project functions independently with its own HTML file.
- Structure and design are fully separated using external style.css files.
- Modern styling techniques like mobile-first queries are applied throughout.
- All required media files are stored locally inside their specific project folders.
